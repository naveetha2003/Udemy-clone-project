# Udemy-clone-project# Udemy Clone (HTML & CSS)

## Project Overview

This project is a simple Udemy Clone website created using HTML and CSS. The purpose of this project is to practice front-end web development concepts such as layout design, styling, and page structure. The website replicates a basic version of the Udemy learning platform with a Home page displaying courses and a Contact page for communication.

This project helped me understand how to structure web pages, style elements using CSS, and organize content in a clean and user-friendly layout. It is a beginner-level front-end project developed as part of my Full Stack Development learning journey.

## Technologies Used

HTML5 – Used to create the structure of the web pages
CSS3 – Used to style the website and design the layout
Flexbox – Used for aligning elements and arranging course cards

## Project Pages

### Home Page

The Home page is the main page of the website and displays the courses similar to an online learning platform.

Features of the Home Page:

* Navigation bar with website title and menu options
* Course section displaying multiple courses
* Course cards with images, title, and short description
* Clean layout designed using CSS
* Flexbox used for arranging course cards in a row

The Home page gives users an overview of the courses available on the platform.

### Contact Page

The Contact page allows users to communicate with the website.

Features of the Contact Page:

* Contact form for user interaction
* Input fields for name, email, and message
* Simple and clean form design
* Styled using CSS for better appearance

This page provides a simple interface for users to send messages or inquiries.

## Features of the Project

* Simple and clean user interface
* Beginner-friendly HTML and CSS structure
* Course card layout similar to Udemy
* Two separate pages connected with navigation
* Organized folder structure for images and files

## Project Purpose

The goal of this project is to practice and improve HTML and CSS skills by building a real-world website layout. This project helped in understanding web page structure, CSS styling techniques, layout design using Flexbox, and creating multi-page websites.

## Future Improvements

In the future, this project can be improved by adding JavaScript for interactive features, responsive design for mobile devices, more pages such as Course Details and About page, and backend integration for user login and course enrollment.
